What is this
You drop in data. Vel scans it, figures out what can be better, tells you how and why, and backs it up with graphs, tables, and full reports. No cloud. No subscription. Runs on hardware you own.

Started as a way to optimize an SC300 build. Turned into a universal engine analysis platform — and eventually a universal optimization engine for any dataset. Every night at midnight it scrapes fresh data, discovers new vehicles and engines, cleans garbage, and rebuilds its knowledge base automatically. It gets smarter while you sleep.
How it works
Hybrid RAG
BM25 keyword search + vector semantic search fused together. Exact engine names are found precisely. Conceptual queries still work. Verified data always wins.
Vehicle-First Data
Every car is scraped by generation and trim. Supra MK4, M3 E46, GT350 — each gets its own entry with engine, power, years, and confidence score.
Self Growing
Midnight pipeline scrapes manufacturers, discovers new models by generation, cleans duplicates with confidence scoring, and rebuilds the index automatically.
Fully Local
Mistral 7B via Ollama. nomic-embed-text for embeddings. No API calls to OpenAI. No data leaves your machine. OpenAI-compatible API endpoint included.
Optimization Engine
Physics-based mod planner. Give it a car and a goal — 500whp, better fuel economy, improved handling — and it returns a ranked mod plan with part names, install locations, known issues, cost, and a phased monthly timeline.
Stack
Mistral 7B Ollama nomic-embed-text LlamaIndex BM25 Hybrid Retriever FastAPI Plotly Dash Open WebUI Textual Tailscale Python systemd Docker CachyOS scipy Optimization Engine
